Calculate the full posterior distribution of your quantity of interest

We have 1000 posterior draws of each model coefficient (taken from two chains à 1000 post-warm-up draws thinned by a factor of 2).

Now, we want to use these to obtain the full posterior distribution of our quantity of interest.

For reference, here is how we calculated our quantity of interest in Lab 1:

# Define value sequences
se_vals <- seq(0, 10, .25)
la_vals <- seq(0, 10, .25)

# Initialize container
pred <- array(NA, dim = c(length(se_vals), length(la_vals), 3L))

# Nested loop for point estimates and confidence bounds
for (i in seq_along(se_vals)) {
  for (j in seq_along(la_vals)) {
    x_star <- c(1, se_vals[i], la_vals[j], se_vals[i] * la_vals[j])
    std_err <- sqrt(as.vector(t(x_star) %*% Sigma %*% x_star))
    pred[i, j, 1] <- x_star %*% b
    pred[i, j, 2] <- pred[i, j, 1] + qnorm(.025) * std_err
    pred[i, j, 3] <- pred[i, j, 1] + qnorm(.975) * std_err
  }
}

Three things will be different this time around:

  1. We do not want point estimates and confidence bounds, but the full posterior. Adjust the dimensions of the container pred accordingly.
  2. Previously, all elements of x_star (except the leading 1) changed at each iteration of the loop. Now, we must hold our background covariates constant at some value, e.g., at their sample mean. Make sure to change the correct elements of x_star at each iteration while holding all others constant.
  3. Previously x_star %*% b was a simple inner product of two \(K\)-length vectors. Now we must multiply an \(S \times K\) matrix (\(S\) draws for \(K\) coefficients) with a \(K\)-length vector \(x^{\ast}\) (remember that the arguments must be conformable).

Let’s give it a try:

...

# Export newly created objects to global environment
export(environment())
# Define value sequences
se_vals <- seq(0, 10, .25)
la_vals <- seq(0, 10, .25)

# Initialize container
pred <- array(NA, dim = c(length(se_vals),
                          length(la_vals),
                          nrow(beta_draws)))

# Get a template vector of sample means
x_tmp <- apply(x, 2, mean)

# Inspect which element holds which predictor
names(x_tmp)

# Get the names or indices of elements you want to change
x_index <- c("se_self", "la_self", "se_self:la_self")

# Nested loop for point estimates and confidence bounds
for (i in seq_along(se_vals)) {
  for (j in seq_along(la_vals)) {
    # Copy and modify x
    x_star <- x_tmp
    x_star[x_index] <- c(se_vals[i], la_vals[j], se_vals[i] * la_vals[j])
    
    ## Store S draws of qoi in pred[i, j, ]
    pred[i, j, ] <- beta_draws %*% x_star
  }
}

# Export newly created objects to global environment
export(environment())

Summarize the posterior distribution of your QOI

The object pred now holds \(S\) posterior draws for the expected value of AfD Support at each combination of se_vals and la_vals.

Let’s summarize the posterior distribution. For each expected values, take the 50%, 2.5% and 97.5% percentiles. This is a quantile-based summary of the posterior distribution that gives us the posterior median and the 95% credible interval.

Hint: You can use apply() on an array, too! Just set the MARGIN argument to those dimensions you would like to retain. But beware: The dimensions of the resulting object may be different from what you would intuitively expect.

pred_summary <- ...

# Export newly created objects to global environment
export(environment())
pred_summary <- apply(pred, 1:2, quantile, c(.5, .025, .975))

# Export newly created objects to global environment
export(environment())
