Data and Methods for Analyzing Comparative Vote Switching Data

Description

The voteswitchR package offers functions for data management and statistical analyses to implement the conceptual framework presented in Cohen, Krause and Abou-Chadi (2021).

voteswichR (development version 0.1.0) consists of the following functions:

For data processing:

  1. build_data_file: Harmonize, impute, map, rake, and aggregate vote switching data

  2. recode_switches: Recode aggregate switching patterns

For model estimation and post-estimation:

  1. run_mavcl: Run the MAVCL model

  2. calculate_pred_error: Calculate the MAEs and RMSEs of MAVCL Predictions

  3. compute_qoi: Compute MAVCL quantities of interest

Package-specific data:

  1. mappings: Mapping of vote choices from surveys to ParlGov and MARPOR IDs.

  2. codebook: Documentations of mappings.

  3. data_guide: Versions, download links, and access details for survey data.
